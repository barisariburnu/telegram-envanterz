# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-24

### Added
- Initial release of Telegram Inventory Bot
- Real-time stock query functionality
- Quick inventory update buttons (+1, +2, +3, -1, -2, -3)
- User authorization system
- Intuitive inline keyboard interface
- Supabase database integration
- Docker support with auto-restart
- Multiple product ID format support
- PM2 ecosystem configuration
- Interactive setup script
- Help command with usage guide
- Main menu navigation
- Product ID direct input support

### Security
- User authorization check on all commands
- Input validation to prevent SQL injection
- Secure `.env` configuration for credentials

### Documentation
- Comprehensive README in English and Turkish
- Contributing guidelines
- MIT License
- Docker deployment instructions
- Setup guides for both local and Docker deployments
